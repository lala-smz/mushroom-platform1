# 时令菌菇盲盒系统用户操作手册

## 1. 系统概述

时令菌菇盲盒系统是一个允许用户抽取包含不同种类菌菇的盲盒的应用，同时提供完整的管理后台功能。本手册将详细介绍系统的使用方法和操作流程。

## 2. 普通用户功能

### 2.1 浏览盲盒

1. **访问盲盒页面**：点击导航栏中的「菌菇盲盒」链接
2. **查看盲盒列表**：页面将显示所有可用的盲盒配置
3. **查看盲盒详情**：点击盲盒卡片进入详情页面，查看盲盒内容和价格

### 2.2 抽取盲盒

1. **选择盲盒**：在盲盒详情页面点击「立即抽取」按钮
2. **确认抽取**：在弹出的确认对话框中点击「确认抽取」
3. **查看结果**：系统将展示抽取过程动画，并显示最终获得的菌菇
4. **查看抽取历史**：在盲盒页面可查看历史抽取记录

### 2.3 数据同步

系统会自动同步盲盒数据，确保您看到的盲盒信息与服务器保持一致。您也可以点击「同步数据」按钮手动触发同步。

## 3. 管理员功能

### 3.1 登录管理后台

1. **访问管理后台**：在浏览器中输入管理后台地址（默认为 `http://localhost:5175/admin`）
2. **输入登录凭证**：使用管理员账号密码登录
3. **进入盲盒管理**：在左侧导航栏中点击「盲盒管理」

### 3.2 盲盒管理

#### 3.2.1 添加新盲盒

1. **点击「添加盲盒」按钮**：在盲盒管理页面顶部
2. **填写盲盒信息**：
   - 盲盒名称
   - 盲盒描述
   - 盲盒价格
   - 盲盒库存
   - 上传盲盒图片
3. **配置盲盒内容**：
   - 选择菌菇类型（常见菌菇或自有商品菌菇）
   - 从下拉列表中选择菌菇
   - 设置每个菌菇的概率
   - 点击「添加」按钮添加到盲盒内容
4. **保存配置**：点击「确定」按钮保存盲盒配置

#### 3.2.2 编辑盲盒

1. **找到要编辑的盲盒**：在盲盒列表中
2. **点击「编辑」按钮**：位于盲盒操作栏
3. **修改盲盒信息**：可以修改所有盲盒参数和内容配置
4. **保存修改**：点击「确定」按钮保存修改

#### 3.2.3 删除盲盒

1. **找到要删除的盲盒**：在盲盒列表中
2. **点击「删除」按钮**：位于盲盒操作栏
3. **确认删除**：在弹出的确认对话框中点击「确定」

#### 3.2.4 查看盲盒统计

1. **点击「统计」按钮**：在盲盒管理页面顶部
2. **查看统计数据**：
   - 销售总额
   - 销售数量
   - 库存总量
   - 各盲盒的详细统计数据
3. **关闭统计对话框**：点击「关闭」按钮

### 3.3 菌菇分类管理

#### 3.3.1 查看菌菇分类

在添加或编辑盲盒内容时，系统会显示菌菇类型筛选器：

1. **选择菌菇类型**：
   - 常见菌菇：系统内置的常见菌菇种类
   - 自有商品菌菇：平台自有商品菌菇
2. **浏览菌菇列表**：筛选后会显示对应类型的菌菇列表

#### 3.3.2 菌菇选择

1. **搜索菌菇**：在搜索框中输入菌菇名称
2. **选择菌菇**：从下拉列表中选择要添加的菌菇
3. **设置概率**：输入该菌菇在盲盒中的出现概率
4. **添加到盲盒**：点击「添加」按钮

## 4. 系统操作指南

### 4.1 常见操作流程

#### 4.1.1 抽取盲盒流程

1. 登录系统
2. 导航到菌菇盲盒页面
3. 选择要抽取的盲盒
4. 点击「立即抽取」按钮
5. 等待抽取动画完成
6. 查看抽取结果
7. 可选择再次抽取或返回

#### 4.1.2 创建盲盒流程

1. 登录管理后台
2. 导航到盲盒管理页面
3. 点击「添加盲盒」按钮
4. 填写盲盒基本信息
5. 配置盲盒内容（选择菌菇和设置概率）
6. 保存盲盒配置
7. 检查盲盒是否显示在前端页面

### 4.2 注意事项

1. **库存管理**：确保盲盒库存充足，避免用户抽取时出现库存不足的情况
2. **概率设置**：合理设置菌菇出现概率，确保用户体验
3. **数据同步**：定期检查数据同步状态，确保前后端数据一致
4. **系统性能**：在高并发场景下，系统会自动优化性能，确保稳定运行

### 4.3 故障排除

#### 4.3.1 数据不同步

- **症状**：前端显示的盲盒数量与后台管理不一致
- **解决方案**：
  1. 点击「同步数据」按钮手动触发同步
  2. 检查网络连接是否正常
  3. 刷新页面后重新尝试

#### 4.3.2 抽取失败

- **症状**：点击抽取按钮后无响应或显示错误
- **解决方案**：
  1. 检查用户登录状态
  2. 检查盲盒库存是否充足
  3. 检查网络连接是否正常
  4. 刷新页面后重新尝试

#### 4.3.3 管理后台操作失败

- **症状**：添加、编辑或删除盲盒操作失败
- **解决方案**：
  1. 检查管理员权限
  2. 检查网络连接是否正常
  3. 检查输入数据是否符合要求
  4. 刷新页面后重新尝试

## 5. 系统要求

### 5.1 浏览器要求

- Chrome 90+ 或其他现代浏览器
- 启用 JavaScript
- 启用 Cookies

### 5.2 网络要求

- 稳定的网络连接

- 桌面端：分辨率 1024x768 及以上
- 移动端：支持响应式布局，适配各种屏幕尺寸
